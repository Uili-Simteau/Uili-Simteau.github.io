TECHNICAL BLOG - Part 2

  • What is Grid Based Design?

    • Grid Based Design is seeing the screen as made up as a flexible grid that you can move objects around in.

  • What is all the hype around responsive websites?

    • Responsive websites respond to the resolution size of the screen the website is being displayed on. This leads to a more consistant experience of a website across different media players.

  • Semantic structure?

    • Semantic structures refers to allocating HTML Tags in a logical or sensible way to the content.

    For example, Headdings 1 to 4 should indicate the relative importance of each headding, or grouping of text.

  • Points of interest I found?
